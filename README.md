# c9-front

Frontend para C9

## Instalación

Para instalar el proyecto, asegúrese de tener Node.js instalado y ejecute el siguiente comando:

```bash
npm install
```

## Ejecución

Para ejecutar el proyecto, ejecute el siguiente comando:

```bash
npm start
```

## Variables de entorno

```bash
cp .env.example .env
```

dentro del archivo .env se debe configurar la url del backend

```bash
PORT=3001

REACT_APP_BACKEND_URL=http://localhost:3000/api/
```
